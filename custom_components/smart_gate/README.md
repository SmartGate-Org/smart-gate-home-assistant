# Smart Gate

Home Assistant custom integration package files for Smart Gate devices.

Use the repository root README for installation and release notes:

- `README.md`
- `docs/INSTALLATION.md`
- `docs/TROUBLESHOOTING.md`
- `docs/LOCAL_API.md`

Supported in v0.4.0:

- SG-Load-Box local HTTP control.
- Zeroconf discovery.
- SW1-SW{channel_count} switch entities.
- Identify button.
- Diagnostic sensors and binary sensors.
- Options Flow for device name and polling interval.
